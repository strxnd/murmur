# Murmur

Open-source system-wide AI dictation.

## Installation

> [!WARNING]
> Murmur is still a very early project. Expect bugs and issues to occur and report if found.

Download the latest Linux package from [GitHub Releases](https://github.com/strxnd/murmur/releases):

- `AppImage` for portable desktop use.
- `.deb` for Debian and Ubuntu-based systems.
- `.rpm` for Fedora, RHEL, and compatible systems.
