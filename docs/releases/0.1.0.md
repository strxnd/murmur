# Murmur 0.1.0

Murmur's first public alpha release is ready for early Linux users who want system-wide AI dictation with local-first speech-to-text.

## Highlights

- System-wide dictation with a global activation hotkey and a compact recording pill.
- First-run setup for microphone selection, local STT model setup, hotkey configuration, and a transcription test.
- Local STT support through bundled `whisper.cpp` and Sherpa ONNX runtimes.
- Downloadable local voice model catalog, including Whisper GGML models and NVIDIA Parakeet ONNX models.
- Built-in dictation modes for default writing, direct voice-to-text, messages, mail, and notes.
- Optional LLM cleanup and rewriting through local providers or explicitly configured cloud providers.
- Linux desktop context and paste automation through the native helper, desktop tools, or clipboard fallback.
- Linux release artifacts for `AppImage`, `.deb`, and `.rpm`, plus unsigned macOS `dmg`/`zip` artifacts for Apple Silicon and Intel. Use `SHA256SUMS.txt` for verification.

## Installation

Download the Linux package for your system from this release:

- Use the `AppImage` for portable desktop use.
- Use the `.deb` package on Debian and Ubuntu-based systems.
- Use the `.rpm` package on Fedora, RHEL, and compatible systems.

Packaged builds include the CPU STT runtimes for the target platform. Voice models are downloaded into the user cache during setup.

## Known Limitations

- This is alpha software. Expect rough edges and report issues with enough detail to reproduce them.
- Linux `x64` is the primary supported packaged target for this release.
- Release packages and checksum manifests are not signed yet.
